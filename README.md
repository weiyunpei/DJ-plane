# DJ-plane
A simple game based on pygame .

The vedio of the game please see here: https://www.bilibili.com/video/BV1pB4y137h3

# DEPENDENCIES
 ```console
   pip install pygame
   ```

# RUN THE GAME
  ```console
   python main.py 
   ```
or run [`main.py`](./main.py) in any IDE
